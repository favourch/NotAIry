## Long Description
NotAIry is a decentralized community notes platform that brings transparency and verification to Web3 and AI content. By combining AI-powered fact-checking with community-driven annotations, we create a trusted layer of verification for crypto projects, smart contracts, and AI-generated content.

### Key Features
- **AI-Powered Verification**: Using Amazon Bedrock's RAG system, NotAIry automatically checks claims against verified sources and on-chain data
- **Community Notes**: Experts and users can add context, warnings, or verification to any Web3 or AI content
- **Reputation System**: Contributors earn reputation through accurate annotations, verified by both AI and community consensus
- **Cross-Platform**: Access notes across platforms (web, Telegram, Discord) through Gaia & Collab.Land integration
- **Secure & Decentralized**: Built on Base L2 with Lit Protocol for secure agent operations

### Use Cases
1. **Smart Contract Verification**: Community notes on contract security, gas optimization, and best practices
2. **AI Model Analysis**: Annotations on model capabilities, limitations, and potential risks
3. **DeFi Protocol Reviews**: Real-time community insights on protocol changes and risk factors
4. **Web3 Project Due Diligence**: Crowdsourced verification of project claims and team credentials

## How it's Made

NotAIry leverages several cutting-edge technologies to create a robust and scalable platform:

### Core Infrastructure
- **Frontend**: Built with SvelteKit for a fast, responsive user experience
- **Blockchain**: Deployed on Base L2 for low-cost transactions and Coinbase integration
- **AI/LLM**: Amazon Bedrock powers our RAG system for automated fact-checking
- **Data Indexing**: The Graph for efficient blockchain data querying and real-time updates

### Key Integrations
1. **Coinbase AgentKit**
   - Manages wallet interactions and onchain actions
   - Provides seamless user authentication
   - Enables fee-free USDC transactions

2. **Lit Protocol Agent Wallet**
   - Secure key management for AI agents
   - Handles user credentials and permissions
   - Manages agent access controls

3. **Gaia & Collab.Land**
   - Enables multi-platform agent deployment
   - Powers social engagement features
   - Provides ready-made agent infrastructure

4. **The Graph**
   - Custom subgraphs for note verification
   - Real-time data aggregation
   - Efficient blockchain data access

### Technical Innovations
- **Hybrid Verification System**: Combines AI fact-checking with community consensus
- **Cross-Platform Architecture**: Unified note system across multiple platforms
- **Scalable Agent Infrastructure**: Distributed agent network for verification tasks
- **Decentralized Reputation**: On-chain tracking of contributor reputation and stake
